OpenSSH для OpenBSD

OpenSSH разрабатывается группой разработчиков OpenBSD, OpenSSH включен в состав операционной системы, начиная с выпуска OpenBSD 2.6. Поэтому ничего не надо устанавливать или добавлять в вашу систему. Тем не менее, возможно вы захотите обновить OpenSSH.

Последний выпуск OpenSSH можно загрузить с ftp.

Следующие версии OpenSSH доступны для OpenBSD:

  • OpenSSH 4.2: 1 сентября 2005

    ftp://ftp.openbsd.org/pub/OpenBSD/OpenSSH/openssh-4.2.tar.gz

    Если вы устанавливаете OpenSSH 4.2 на OpenBSD 3.5, то вам нужен следующий патч:
    ftp://ftp.openbsd.org/pub/OpenBSD/OpenSSH/openbsd35_4.2.patch.

    # cd /usr/src/usr.bin
# tar xvfz .../openssh-4.2.tar.gz
# cd ssh
# patch -p0 < /path/to/openbsd35_4.2.patch
# make obj
# make cleandir
# make depend
# make
# make install
# cp ssh_config sshd_config /etc/ssh
    и перезапустите sshd.

  • OpenSSH 4.0: 9 марта 2005

    ftp://ftp.openbsd.org/pub/OpenBSD/OpenSSH/openssh-4.0.tgz

    Если вы устанавливаете OpenSSH 4.0 на OpenBSD 3.5, то вам нужен следующий патч:
    ftp://ftp.openbsd.org/pub/OpenBSD/OpenSSH/openbsd35_4.0.patch.

    # cd /usr/src/usr.bin
# tar xvfz .../openssh-4.0.tgz
# cd ssh
# patch -p0 < /path/to/openbsd3x_4.0.patch
# make obj
# make cleandir
# make depend
# make
# make install
# cp ssh_config sshd_config /etc/ssh
# mkdir /var/empty
    и перезапустите sshd. Для поддержки в sshd(8) возможности разделения привилегий на некоторых старых выпусках OpenBSD в систему должны быть добавлены новый пользователь и группа sshd. Добавьте следующее, используя vipw(8): 
    sshd:*:27:27::0:0:sshd privsep:/var/empty:/sbin/nologin

    Добавьте следующее в /etc/group:

    sshd:*:27:
_sshagnt:*:34:
