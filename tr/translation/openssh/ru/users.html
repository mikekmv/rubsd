Системы в которых используется OpenSSH

Известно, что перечисленные операционные системы и продукты интегрировали OpenSSH в базовую поставку.

Список составлен в хронологическом порядке, системы которые раньше интегрировали OpenSSH перечислены первыми.

  • FreeBSD
  • Suse Linux
  • Redhat Linux
  • Mandrake Linux
  • BSDi BSD/OS
  • NetBSD
  • Computone
  • Conectiva Linux
  • Slackware Linux
  • Caldera OpenLinux
  • Stallion
  • Rock Linux
  • Cygwin
  • e-smith server и gateway
  • Engarde Linux
  • Mac OS X Version 10.1
  • HP Procurve Switch 4108GL и 2524/2512
  • IBM AIX
  • Gentoo Linux
  • Gwynux/Toadware Linux
  • Sun Solaris 9 (называется SunSSH)
  • SmoothWall
  • IPCop
  • SGI Irix
  • ThinLinc
  • Nokia IPSO
  • Cisco CSS11500 series content services switches
  • Cisco SN 5400 series storage routers
  • TopLayer IDS balancers
  • NTI SSH Serial Port Switch
  • Bluecoat (бывшый Cacheflow) Proxy SG

Если здесь не указана система, которая содержит OpenSSH в базовой поставке, пожалуйста сообщите об этом нам. (Предполагается, что OpenSSH включена по умолчанию, например на CD или встроена в систему, и не требует того, чтобы пользователь её самостоятельно устанавливал).
