Системы в которых используется OpenSSH

Известно, что перечисленные операционные системы и продукты интегрировали OpenSSH в базовую поставку.

Список составлен в хронологическом порядке, системы которые раньше интегрировали OpenSSH перечислены первыми.

  • OpenBSD
  • FreeBSD
  • BSDi BSD/OS
  • NetBSD
  • Computone
  • Stallion
  • Cygwin
  • e-smith server и gateway
  • Mac OS X начиная с версии 10.1
  • HP Procurve Switch 4108GL and 2524/2512
  • IBM AIX
  • Sun Solaris, начиная с версии 9 (называется SunSSH)
  • SmoothWall
  • IPCop
  • SGI Irix
  • ThinLinc
  • Nokia IPSO
  • Cisco CSS11500 series content services switches
  • Cisco SN 5400 series storage routers
  • TopLayer IDS balancers
  • NTI SSH Serial Port Switch
  • Bluecoat (ранее назывались Cacheflow) Proxy SG
  • Novell NetWare
  • Digi CM Console Servers
  • Alcatel OmniSwitch
  • Dell PowerConnect L2 и L3 Switches
  • HP-UX (known as HP-UX Secure Shell)
  • Packeteer PacketShaper 6.0 и новее.
  • Juniper Networks JUNOS
  • All Linux systems

Если в этом списке нет системы, которая по умолчанию поставляется с OpenSSH, то, пожалуйста, сообщите об этом нам. (Подразумевается, что OpenSSH включен по умолчанию в дистрибутив системы или находится на CD и не требует, чтобы пользователь самостоятельно его устанавливал).
