Mac OS

Следующие клиенты рекомендованы для взаимодействия с OpenSSH из Mac OS. Примечание: в дистрибутив Mac OS X OpenSSH уже включен.

  • NiftyTelnet 1.1 SSH только SSH1 реализация, программа в стиле scp. Написана Jonas Wallden.

    "NiftyTelnet 1.1 SSH r3 это расширенная версия приложения NiftyTelnet 1.1 от Chris Newman, в которую добавлена поддержка шифрованных терминальных сессий, с использованием протокола SSH (Secure Shell). Пожалуйста прочитайте вложенный Readme файл перед использованием этой версии."

  • MacSSH является реализацией только SSH2.

    "MacSSH это модифицированная версия BetterTelnet с поддержкой SSH2. [...] Единственный SSH2 клиент для Mac OS, который я смог найти, был коммерческим продуктом и стоил больше $100 и он "падал" на моем Mac, когда закрывал сессию... Посклольку лучше всего делать вещи самому, я создал MacSSH."

  • Fugu это реализация SFTP и SCP для Mac OS X.

    "Fugu SSH это графическая оболочка для программы OpenSSH Secure File Transfer (SFTP) для Mac OS X. Fugu позволит вам использовать преимущества безопасности SFTP без жертвования удобством использования, которые вы найдете в GUI. Fugu так же включает поддержку SCP для обмена файлами, и возможность создания защищенных туннелей через SSH."

  • Cyberduck это SFTP и FTP браузер для Mac OS X.

    "Cyberduck это SFTP (SSH Secure File Transfer) и FTP браузер с открытыми исходными текстами, распространяется под GPL лицензией. Имеется поддержка множественных подключений. [...] Так же имеется поддержка основных системных технологий таких как Keychain и Rendezvous. "
