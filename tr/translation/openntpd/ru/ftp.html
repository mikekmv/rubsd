FTP и HTTP зеркала OpenNTPD

Базовый код OpenNTPD 3.6 выпущен 2 ноября 2004 года.

Базовый код разработан специально для OpenBSD, дабы сделать его простым, чистым, минимальным и удобным для аудита. Отдельная группа разработчиков преобразует OpenNTPD в портируемую версию, которая работает на других операционных системах.

Повторяем: Для других операционных систем используйте портируемую версию!
