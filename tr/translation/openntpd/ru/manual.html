Документация

На Web сайте проекта OpenBSD доступны man страницы для последней версии OpenNTPD.

Они описывают последний выпуск OpenNTPD для разработчиков.

  • ntpd - сервис NTP (Network Time Protocol)
  • ntpd.conf - описание формата конфигурационного файла OpenNTPD
Протокол NTP, реализованный в OpenNTPD, описан в RFC 1305 и RFC 2030. Эталонная реализация и другая документация по протоколу NTP доступна с сайта ntp.org.
