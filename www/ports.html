Неофициальные порты

На этой страничке дано краткое описание неофициальных портов, созданных в рамках проекта OpenBSD.ru. Дерево неофициальных портов можно получить из CVS репозитория OpenBSD.ru.

Настройка доступа к репозиторию: 

# vi /etc/ssh/ssh_config
Host anoncvs.openbsd.ru
        Port 2022
Получение дерева неофициальных портов: 
# cd /usr/ports
# cvs -d anoncvs@anoncvs.openbsd.ru:/rucvs get -PA mystuff
Обновление дерева: 
# cd /usr/ports/mystuff
# cvs up -PAd

Для работы с неофициальными портами требуется наличие официального дерева портов в /usr/ports. Дерево неофициальных портов должно располагаться в каталоге /usr/ports/mystuff. Порты рассчитаны на использование в OpenBSD 4.4 и новее. Для ранних версий OpenBSD потребуется обновление дерева портов и src/usr.sbin/pkg_add.
