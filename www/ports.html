Неофициальные порты

На этой страничке дано краткое описание неофициальных портов, созданных в рамка проекта OpenBSD.ru. Дерево неофициальных портов можно получить из репозитория OpenBSD.ru.

Настройка доступа к репозиторию: 

# vi /etc/ssh/ssh_config
Host anoncvs.openbsd.ru
        Port 2022
Получение дерева неофициальных портов: 
# cd /usr/ports
# cvs -d anoncvs@anoncvs.openbsd.ru:/cvs get -PA mystuff
Обновление дерева: 
# cd /usr/ports
# cvs up -PAd
Для работы с неофициальными портами требуется наличие официального дерева портов в /usr/ports. Дерево неофициальных портов должно располагаться в каталоге /usr/ports/mystuff. Порты расчитаны на использование в OpenBSD-current. Для ранних версий OpenBSD потребуется обновление дерева портов и src/usr.sbin/pkg_add.
