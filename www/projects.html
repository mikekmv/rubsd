Проекты

На этой страничке собраны проекты, созданные участниками OpenBSD.ru для облегчения собственной жизни. Надеемся, что они окажутся полезными не только нам. ;)

Для доступа к репозиторию с проектами можно воспользоваться либо AnonCVS, либо CVSweb.
Параметры доступа через AnonCVS:

CVSROOT=anoncvs@anoncvs.openbsd.ru:/rucvs
CVS_RSH=ssh

cat >> ~/.ssh/config << EOF
Host anoncvs*.openbsd*
        Port 2022
EOF


Название Описание Где взять Куда слать патчи Статус
aylet Порт проигрывателя файлов музыкального сопроцессора ZX Spectrum AY. ports/audio/aylet grange@openbsd.ru Версия 0.2. Не обновляется.
xzx-pro Порт эмулятора ZX Spectrum. ports/emulators/xzx-pro grange@openbsd.ru Версия 4.1. Не обновляется.
mnogosearch Порт поискового движка для сайтов. ports/www/mnogosearch grange@openbsd.ru Версия 3.1.20. Не обновляется.
mail.buhal Замена для mail.local. Позволяет сохранять приходящую почту в формате maildir. src/mail.buhal support@openbsd.ru Версия 1.0beta. Не обновляется.
cnupm Система для подсчета IPv4/IPv6 трафика. src/cnupm support@openbsd.ru Версия 3.11. Финиш.
ipstatd Демон статистики IPv4 трафика. src/ipstatd support@openbsd.ru Документация отсутствует. Не обновляется. Работает.
tunneld Предназначен для создания IP tunnel между двумя хостами/сетями, физически находящимися в разных сетях с использованием tun интерфейса. Поддерживает трансляцию адресов. src/tunneld support@openbsd.ru Не обновляется.
scripts Немного простых скриптов для обновления и поддержки системы. src/scripts grange@openbsd.ru Поддерживается.
if_acct Cnupm-подобная считалка трафика для PF на уровне кернела. src/if_acct support@openbsd.org Не поддерживается.
ipflow Считалка трафика, использующая BPF на уровне ядра. src/ipflow form@pdp-11.org.ru Версия 1.5.
hideproc Управляемый прятатель чужих процессов src/hideproc form@pdp-11.org.ru Версия 2.0.
bcrelayd Broadcast relay daemon. Позволяет сетевым игрушкам и другим программам, ищущим свои сервера по broadcast, находить их за пределами локальной сети. src/bcrelayd form@pdp-11.org.ru Без версии.
sshwatchd Простая программа для блокировки доступа с адресов, пытающихся подобрать пароли на SSH методом перебора. src/sshwatchd support@openbsd.ru Без версии.
