Примеры

Данные примеры составлены из ответов на некоторые вопросы, касающиеся программирования в OpenBSD. Все программы, используемые в данных примерах могут быть свободно использованы в любых целях без всяких ограничений.

grabcons.c, Makefile Перехват консольных сообщений
dynamic.c, module.c, module.h, Makefile Динамическая загрузка модулей
kevent.c, Makefile Использование kevent
eventsrv.c, Makefile Использование libevent и getaddrinfo в серверных программах
milter.c, Makefile Шаблон для написания Milter фильтров к sendmail и postfix
tree.c, Makefile Пример использования RB деревьев
signpkt.c, Makefile Пример использования HMAC-SHA1 для подписывания сетевых пакетов
 
examples.tar.gz Все вышеперечисленные примеры
