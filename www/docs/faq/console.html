Как сделать, чтобы экран автоматически очищался при выходе пользователя из системы?

Для того чтобы экран автоматически очищался при выходе пользователя из системы, следует добавить в файле gettytab в запись, соответствующую текстовой консоли, строчку cl=\E[H\E[J. Пример: 

P|Pc|Pc console:\
        :np:sp#9600:cl=\E[H\E[J:

Не могу переключиться между консолями в VMWare. Что делать?

По умолчанию VMWare использует комбинацию клавиш Ctrl+Alt для освобождения курсора мыши, что мешает обычному переключению между консолями с помощью Ctrl-Alt-Fn. Для переключения между консолями следует нажать Ctrl-Alt-Пробел, после чего, не отпуская клавиш Ctrl и Alt, нажать нужную клавишу Fn.
