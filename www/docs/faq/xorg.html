При вставке текста с помощью мыши в xterm вместо русских букв появляется #. Как поправить?

Для того, чтобы вставка русских букв в xterm работала нормально, нужно запускать его командой xterm -k8 или добавить в файл .Xdefaults следующую строчку: 

XTerm*allowC1Printable: true

Как включить прокрутку/перемещение курсора с помощью колеса мыши в xterm?

Для того, чтобы с помощью колеса мыши можно было перемещать курсор (в текстовых редакторах) или прокручивать текст (например в программе less) следует добавить следующие строки в файл .Xdefaults: 

XTerm*vt100.translations: #override \n\
	<Btn4Down>, <Btn4Up>: string(0x1b) string("OA") \n\
	<Btn5Down>, <Btn5Up>: string(0x1b) string("OB")

При использовании xdm наблюдаются проблемы с клавиатурой. В чем проблема?

Наиболее вероятная причина - запуск X сервера на виртуальной консоли, на которой активен процесс getty. Следует убедиться, что для консоли, используемой в файле /etc/X11/xdm/Xservers не запускается процесс getty в файле /etc/ttys. Следует обратить внимание, что виртуальные консоли в файле /etc/X11/xdm/Xservers начинаются с vt01 (соответствует ttyC0).
