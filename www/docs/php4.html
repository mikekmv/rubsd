Устанавливаем PHP4

Данный документ представляет собой пошаговое руководство по установке PHP4 из коллекции пакетов на OpenBSD 3.3. Установка включает в себя инсталляцию базового пакета и различных дополнительных модулей, которые не зависят друг от друга. Необходимые файлы могут быть найдены на ftp или OpenBSD CD.

Сначала установим базовый пакет php4-core-4.2.3.tgz и используем команду phpxs для подключения PHP4 к Apache. Скопируем и при необходимости настроим конфигурационный файл php.ini. 

	# pkg_add php4-core-4.2.3.tgz
	# /usr/local/sbin/phpxs -s
	# cp /usr/local/share/doc/php4/php.ini-recommended /var/www/conf/php.ini
	# vi /var/www/conf/php.ini

Далее, установим набор классов для общих задач (php4-pear-4.2.3.tgz) и дополнительные пакеты-модули для поддержки MySQL и IMAP (php4-mysql-4.2.3.tgz, php4-imap-4.2.3.tgz). Активизируем их командой phpxs. 

	# pkg_add php4-pear-4.2.3.tgz
	..
	# pkg_add php4-mysql-4.2.3.tgz
	# /usr/local/sbin/phpxs -a mysql
	..
	# pkg_add php4-imap-4.2.3.tgz
	# /usr/local/sbin/phpxs -a imap

Проверим работу PHP: 

	# echo '<?php phpinfo() ?>' > /var/www/htdocs/test.php
	# lynx http://localhost/test.php

В результате, мы должны получить информацию о работе модуля PHP и Apache.
