IRC

   Существует канал #openbsd на сервере irc.openbsd.ru. Тематика канала - обсуждение OpenBSD и вопросов OpenBSD.ru. Язык канала - русский, кодировка - cp1251 (это вызвано тем, что большинство IRC клиентов, работающих под Windows, не поддерживают перекодировку).

Для работы с каналом из под UNIX систем, проще всего воспользоваться программой ircII, которая поддерживает перекодировку koi8 <> cp1251. Для работы из под Windows, можно воспользоваться любым IRC клиентом, который не против русских букв. Для работы из под OS/2 лучше всего воспользоваться программой OpenChat/2.

Чтобы включить перекодировку для:

  • ircII

    Создайте в домашнем каталоге файл .ircrc содержащий строчки: 
      set eight_bit_characters on
set translation RUSSIAN_WIN

  • irssi

    Для настройки cp1251 в irssi вам необходимы файлы перекодировки от ircII или BitchX. Запустите irssi и введите команды: 
      /set translation /usr/local/share/irc/translation/RUSSIAN_WIN
/save
    Команда /save сохранит настройки.
